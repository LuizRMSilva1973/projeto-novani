# Site da Maria Gabriela Mantovani

Landing page estática inspirada no estilo neon/escuro do case “Blinky”, feita para apresentar os serviços de marketing veterinário da Maria Gabriela Mantovani.

## Destaques
- Paleta escura com acentos azul/roxo e brilho suave em botões e cards.
- Hero fullscreen com blob animado, badge e métricas rápidas.
- Cards de serviços com chips, micro-CTA e seções “Quem sou”, “Diferenciais” e “Contato”.
- Layout responsivo simples para desktop e mobile.

## Como usar
1. Baixe ou clone este repositório.
2. Abra `index.html` no navegador (é um site estático, sem build).

## Estrutura
```
.
├── index.html   # Página principal e estilos inline
├── img/         # Imagens utilizadas no hero e demais seções
└── video/       # Pasta reservada para vídeos (se necessário)
```

## Personalização rápida
- Troque a foto do hero em `img/IMG_0043 copiar (3).jpg` mantendo proporção horizontal.
- Atualize o Instagram real na seção “Contato direto”.
- Ajuste cores no bloco `:root` em `index.html` para novas variações de acento.
